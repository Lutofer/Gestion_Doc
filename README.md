#  Sistema de Gestión de Restaurantes

##  Introducción
En el sector gastronómico, la eficiencia en la gestión de los procesos internos es clave para brindar un servicio de calidad. Sin embargo, muchos restaurantes aún presentan problemas como la falta de organización, demoras en la atención al cliente, errores en los pedidos y un control ineficiente del inventario.  

El presente proyecto tiene como finalidad desarrollar un sistema de gestión de restaurantes que permita optimizar estos procesos mediante el uso de herramientas tecnológicas. A través de este sistema, se busca mejorar la administración del negocio, reducir errores humanos y agilizar la atención al cliente.

---

##  Objetivo General
Desarrollar un sistema de gestión para restaurantes que permita optimizar los procesos operativos, mejorando la organización, reduciendo tiempos de atención y facilitando la administración de recursos.

---

##  Objetivos Específicos
- Implementar un módulo de gestión de pedidos que permita registrar, visualizar y controlar los pedidos de manera eficiente.  
- Desarrollar un sistema de control de inventario que facilite el seguimiento de productos disponibles.  
- Gestionar usuarios dentro del sistema, asignando roles y permisos según sus funciones.  
- Automatizar procesos manuales para reducir errores y mejorar la productividad.  
- Mejorar la experiencia del cliente mediante una atención más rápida y organizada.  

---

##  Justificación
La implementación de este sistema surge como respuesta a las problemáticas comunes en los restaurantes, tales como la desorganización en los pedidos, la demora en la atención y la falta de control en el inventario.  

Al digitalizar estos procesos, se logra una mejor gestión de la información, lo que permite tomar decisiones más acertadas y en menor tiempo. Además, se contribuye a mejorar la calidad del servicio ofrecido, incrementando la satisfacción del cliente y la competitividad del negocio en el mercado.

---

##  Funcionalidades del Sistema
-  Gestión de inventario  
-  Registro y control de pedidos  
-  Administración de usuarios  
-  Organización de información para reportes  
- Optimización de tiempos de atención  

---

##  Tecnologías Utilizadas
- **Lenguaje de programación:** Java  
- **Framework:** Spring Boot  
- **Base de datos:** MySQL  
- **Control de versiones:** Git y GitHub
